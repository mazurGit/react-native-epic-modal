# react-native-epic-modal

A flexible and powerful modal manager for React Native.

## Installation

```sh
npm install react-native-epic-modal
```

## Usage


```js
import { multiply } from 'react-native-epic-modal';

// ...

const result = await multiply(3, 7);
```
